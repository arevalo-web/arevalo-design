NOVEMBER 2025 TRANSIT SYSTEM

THE PROBLEM

Las Vegas requires an underground rapid transit system that is easy to navigate for both tourists and locals. With five new routes being established, the challenge was to design a modern and cohesive transit diagram. This project focuses on simplifying complex city infrastructure into a clear and functional visual guide.

THE SYSTEM

The system uses a vibrant yet functional color palette to ensure each route is distinct and easy to follow. By prioritizing high-contrast colors and a mathematical 45-degree grid, the layout achieves a balance between technical precision and approachability. This modular design is built to be easily replicable as the transit network grows.

THE RESULT

A clean sans-serif typeface combined with simplified iconography ensures maximum legibility in high-traffic environments. The final design aids in rapid wayfinding and passenger flow through the city. This system transforms the underground experience into a predictable and efficient journey for every passenger.
